# API Stability Matrix
| Module       | Status       | Support Horizon | Breaking Change Policy      |
|--------------|--------------|-----------------|-----------------------------|
| `api/`       | Stable       | 3 Years         | Only in Major Versions      |
| `internal/`  | Unstable     | None            | Anytime without notice      |
| `experimental/`| Beta       | 6 Months        | Allowed with 1 minor notice |
| `deprecated/`| Sunset       | 1 Year          | Will be removed in next Major |
